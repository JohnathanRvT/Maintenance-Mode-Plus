== Description ==

Tired of the default WordPress maintenance page? **Maintenance Mode Plus** helps you create a stunning, professional-looking "Coming Soon" or "Under Construction" page in just a few clicks.

This plugin will allow you to:
*   Display a custom message to your visitors.
*   Add your brand's logo.
*   Set a full-screen background image.
*   Add a countdown timer.
*   (Coming Soon) Capture leads with a newsletter signup form.

Logged-in administrators can still access the full website, making it easy to perform updates while visitors see the maintenance page.

== Installation ==

1. Upload `maintenance-mode-plus.zip` via the 'Plugins > Add New' screen in WordPress.
2. Activate the plugin through the ‘Plugins’ menu.
3. Navigate to the 'Maintenance Plus' settings page in your dashboard to configure the page.
